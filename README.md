# Map using map-gl

## Purpose

Learning to display map on an html page using react-map-gl

## Dependencies

```javascript
npm install react-map-gl
```

## Pre-requisites

React and Mapbox access token.
In .env file add
REACT_APP_MAPBOX_KEY=YOUR_ACESS_TOKEN
